# Tom ramassage de feuilles / Tom's Leaf Removal Service - Site Web

## Contenu
- index.html (page principale, FR par défaut, toggle EN)
- merci.html (page de remerciement bilingue)
- styles.css (styles et thème)
- script.js (logique du site, bascule de langue, mailto, redirection merci)
- assets/ (logo, images, icônes)

## Mise en ligne avec GitHub Pages

1. Créez un compte sur https://github.com si ce n'est pas déjà fait.
2. Créez un nouveau dépôt (Public) pour votre site.
3. Cliquez sur **Add file → Upload files** et glissez **tous les fichiers** contenus dans ce dossier (pas le dossier lui-même).
4. Validez avec **Commit changes**.
5. Allez dans **Settings → Pages**.
6. Dans **Source**, choisissez la branche `main` et le dossier `/ (root)` puis **Save**.
7. Après quelques secondes, votre site sera en ligne à l'adresse :
   https://<votre-pseudo>.github.io/<nom-du-repo>/
